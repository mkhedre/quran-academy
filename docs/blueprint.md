# **App Name**: Buruj Rewards Landing

## Core Features:

- Sticky Header: A header that transitions from transparent to solid on scroll, displaying the logo, navigation links (Programs, Rewards System), and a prominent 'Join Now' button.
- Hero Section: A two-column layout featuring a headline ('Empowering Souls Through Quran & Rewards' in Arabic), descriptive subtext ('Learn, Memorize, and get instantly rewarded for every Ayah' in Arabic), and an image placeholder of the Quran with a glow effect.
- Reward Calculator: Interactive slider allowing users to input the number of 'Ayahs Memorized' (in Arabic) and view the corresponding 'Expected Reward' in a digital wallet animation.
- Program Features Grid: Three-card grid displaying key program features, namely: 'Live Sessions', 'Precision Recitation (Friday SARD)', and 'Cash Incentives' (in Arabic).
- Leaderboard Section: Displays 'Top Reciters of the Week' alongside their rewards to foster competition and showcase successful participants. All displayed in Arabic.
- Registration Form: A focused CTA section featuring a clean registration form with fields for 'Name', 'Whatsapp', and 'Goal'. Labels and instructions provided in Arabic.
- WhatsApp Support: Persistent floating WhatsApp button to allow direct engagement with support in Arabic.

## Style Guidelines:

- Primary color: Deep Navy (#0A2647) to represent professionalism and trust.
- Secondary color: Soft Gold (#D4AF37) as an accent to highlight key elements.
- Background color: Clean White (#F8FAFC) to create a pristine, uncluttered layout.
- Headline font: 'Amiri' (serif) for titles and headings, conveying a sense of tradition and sophistication. Note: currently only Google Fonts are supported.
- Body font: 'Cairo' (sans-serif) for body text to ensure readability and a modern aesthetic. Note: currently only Google Fonts are supported.
- Utilize elegant, minimalist icons to represent features and benefits.
- Maintain a clean and structured layout, ensuring information is easily accessible and scannable.
- Employ subtle 'reveal on scroll' animations using Framer Motion to create a dynamic and engaging user experience.
